Вариант 16

a) Разработайте абстрактный тип данных - множество set. Дополнительно
перегрузить следующие операции:
() - конструктор множества (в стиле конструктора для множественного типа,
исключающего дубликаты);
+ - объединение множеств;
+ <= - сравнение множеств.

b) Разработайте иерархию классов геометрические фигуры (приватные
члены сконструировать самостоятельно). Добавьте статические члены по
подсчету суммарной площади фигур. Разработайте не менее 2-х виртуальных
функций.

c) Создайте шаблонный класс множества геометрических фигур,
множества действительных чисел. Продемонстрируйте работу класса и все его
операции.
